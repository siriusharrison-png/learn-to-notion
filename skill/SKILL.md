# 添加知识到知识库（Notion 版）

当用户使用 `/learn [术语]` 或说"把 xxx 加入知识库"时，执行以下步骤：

## 第一步：获取术语
- 术语内容来自 `$ARGUMENTS`
- 如果没有提供术语，询问用户想要添加什么

## 第二步：搜索解释
1. 用简单易懂的语言搜索该术语的定义
2. 找到一个适合初学者的学习链接（优先：维基百科中文、MDN 中文、知乎专栏等）

## 第三步：判断分类
根据术语内容判断属于哪个分类：
- **技术概念**：编程、软件、代码相关
- **设计相关**：用户体验、交互设计、视觉设计相关
- **其他**：不属于以上两类

## 第四步：双写到 Notion 和本地

使用 Bash 工具运行以下命令（注意替换参数）：

```bash
cd ~/Desktop/learn-to-notion && node scripts/add-knowledge.js "术语名" "分类" "简单解释" "学习链接"
```

参数说明：
- 术语名：用户提供的术语
- 分类：技术概念 / 设计相关 / 其他
- 简单解释：一句话，不超过 30 字
- 学习链接：找到的学习资源 URL

## 第五步：反馈结果

用以下格式告诉用户：

```
已添加到知识库！

术语：[术语名]
分类：[分类名]
解释：[简单解释]
链接：[学习链接]

✅ 已同步到 Notion
✅ 已备份到本地
```

## 注意事项
- 解释要简单易懂，避免专业术语
- 如果术语已存在于知识库中，提醒用户
- 链接优先选择中文资源
- 如果 Notion 写入失败，仍然尝试写入本地
